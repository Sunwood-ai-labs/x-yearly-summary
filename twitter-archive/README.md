# Twitter Archive フォルダ

このフォルダは、Twitterアーカイブデータを保存するために用意されています。

## フォルダ構造

```
twitter-archive/
├── 2024/          # 2024年のTwitterアーカイブ
│   └── (ここにTwitterのアーカイブファイルを配置してください)
├── 2025/          # 2025年のTwitterアーカイブ
│   └── (ここにTwitterのアーカイブファイルを配置してください)
└── README.md      # このファイル
```

## Twitterアーカイブのダウンロード方法

1. Twitterにログイン
2. 設定とプライバシー > アカウント > データのアーカイブをダウンロード
3. リクエストを送信し、ダウンロードが可能になったら通知が届きます
4. ダウンロードしたZIPファイルを解凍
5. 解凍したフォルダの内容を、該当する年のフォルダ(2024または2025)に配置してください

## アーカイブデータの主な内容

- `tweets.js` - ツイートデータ
- `profile.js` - プロフィール情報
- `followers.js` - フォロワー情報
- `following.js` - フォロー情報
- その他のメディアファイルやデータ

## 注意事項

- アーカイブデータには個人情報が含まれる可能性があります
- 公開リポジトリにプッシュしないよう注意してください
- `.gitignore`でアーカイブフォルダが除外されていることを確認してください
